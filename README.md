# 多伦多炮王 · Snapfire

DOTA2 同人单页小游戏:Snapfire 驻守多伦多 CN Tower 十年,一门加农炮,初始十发炮弹。

- **玩法**:按住拖拽瞄准蓄力(愤怒的小鸟式),实时抛物线预览落点与溅射圈,松手开炮
- 命中**女英雄** → 收获「X代目」,头像收进右侧收集栈,并回补 1 发弹药
- 命中**男英雄** → 当场塌房,直接结算
- 空炮 → 白白浪费一发;弹药打光 → 功成身退
- **结算**:「多伦多炮王编年史」时间轴,第 1 发 = 2016 年,每发一年,同年命中的代目向上堆叠,附称号(纯爱战神 → 一代炮王 → 永动机)

纯静态 `index.html` + `game.js` + `assets/`,无构建、无依赖,直接部署 GitHub Pages。逻辑核心与渲染分离,`game.js` 兼容 Node 可无头验证。

## 本地预览

浏览器直接打开 `index.html`。URL 加 `?seed=42` 可固定随机种子。

## 无头回归

```bash
node tools/test.js
```

覆盖:预览弹道与实际落点一致性、弹药收支、收集/塌房/弹尽三种结局分支、复合标记、中文序数。

## 素材

全部像素素材为 AI 生成(品红底色 + `tools/prepare_assets.py` 连通域切分抠图)。英雄阵容 24 位:女 16 / 男 8。
